## 网站功能说明

这是一个专注于微积分学习的交互式平台，旨在通过可视化和互动方式帮助用户理解微积分核心概念。

### 1. 核心学习内容

#### 积分概念辨析
- 详细解释定积分和不定积分的定义
- 直观展示两者的几何意义
- 提供丰富的应用示例

#### 微积分基本定理
- 系统阐述微积分基本定理的两个部分
- 强调其在微积分发展中的重要意义
- 结合图形说明定理的实际应用

#### 公式宝典
- 整理常用积分公式
- 详细讲解积分方法（换元积分法、分部积分法等）
- 提供公式的应用场景说明

### 2. 交互式元素

#### 动画演示
- 通过动态动画直观展示微积分基本定理的几何意义
- 支持播放、暂停、重新开始等控制功能
- 动画过程配有详细解说

#### 定积分可视化
- 支持用户自定义函数表达式
- 可调整积分上下限
- 动态绘制相应的定积分图形
- 实时计算并显示积分结果

#### AI聊天助手
- 智能解答用户关于微积分的问题
- 提供概念解释、计算步骤等详细内容
- 支持数学表达式输入（如 "计算∫x²dx"）
- 可保存对话历史供用户回顾

### 3. 辅助学习资源

#### 历史长廊
- 介绍从阿基米德到柯西等数学家的贡献
- 展示积分概念的发展历程

#### 思维启迪
- 探讨积分在现实世界中的广泛应用

- 提供实际问题的解决方案

#### 快捷提问
- 提供常见问题的快捷访问按钮

## 使用指南

### 导航浏览
- 通过顶部导航栏可快速跳转到不同学习模块
- 各模块之间设有便捷的跳转链接
- 支持面包屑导航，方便用户了解当前位置

### AI助手交互
1. 在底部输入框中输入微积分相关问题
2. 点击发送按钮提交问题
3. 可点击快捷提问按钮获取常见问题的答案
4. 支持LaTeX格式的数学表达式输入
5. 对话内容会实时显示在聊天窗口中

### 动画演示
1. 进入微积分基本定理模块
2. 点击"播放动画"按钮查看动态演示
3. 使用"重新开始"按钮重置动画
4. 通过"暂停/继续"按钮控制动画播放状态
5. 配合动画解说加深理解

### 定积分可视化
1. 进入定积分可视化模块
2. 在函数输入框中输入自定义函数（如 x^2, sin(x) 等）
3. 调整积分上下限的滑块或输入具体数值
4. 点击"更新"按钮查看相应的定积分图形
5. 观察图形变化，理解积分的几何意义
